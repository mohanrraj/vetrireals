FRAMER MOTION ANIMATIONS:
Used throughout components for:
- Scroll-triggered animations (useInView)
- Hover effects (whileHover)
- Tap effects (whileTap)
- Page transitions (AnimatePresence)

COMMON ANIMATION PATTERNS:

1. FADE + SLIDE UP:
initial={{ opacity: 0, y: 50 }}
animate={{ opacity: 1, y: 0 }}
- Starts invisible and 50px below
- Fades in while sliding up

2. FADE + SLIDE LEFT:
initial={{ opacity: 0, x: -50 }}
animate={{ opacity: 1, x: 0 }}
- Starts invisible and 50px left
- Fades in while sliding right

3. SCALE ON HOVER:
whileHover={{ scale: 1.05 }}
- Grows to 105% on hover
- Smooth transition

4. LIFT ON HOVER:
hover:-translate-y-2
- Moves up 8px on hover
- Creates floating effect
